# OpenBridge - Remote CLI Bridge

## Installation

### One-Line Install
```bash
curl -fsSL https://get.openbridge.dev | bash
```

### Manual Install
```bash
pip install openbridge
openbridge setup
openbridge start
```

## Docker

```bash
docker-compose up -d
```

## Quick Start

1. Install OpenBridge
2. Run `openbridge setup` to configure
3. Start with `openbridge start`
4. Message your bot from Telegram/Discord

## Commands

- `openbridge setup` - Interactive configuration
- `openbridge start` - Start server
- `openbridge status` - Check status
- `openbridge version` - Show version
